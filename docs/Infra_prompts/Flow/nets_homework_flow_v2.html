Global Forbids

Two rules.
Apply everywhere.

These bind every section of the flow. Textbook fidelity is non-negotiable; everything else is built on top of it.

FORBID 01
NETS pretending to replace the official textbook.
FORBID 02

Each Section

Role and forbids,
one per section.

Every flow stage has a defined role, a feel, and the things it must never become.

Learning Section
Case-Based Preview
The NETS learning layer built from textbook material. Teaches through a real-life case, not a passive reading blob.
Includes
Textbook summary · story/case setup · real-life framing · decision checkpoints · SVG/animation moments · AI feedback on decision-making · thought-process summary.
Feel
I just learned this idea, now I immediately use it in a realistic situation.
  • Case Study that loses the original textbook concept.
  • Case Study without checkpoint-based learning.
  • Case Study where the student is not the decision-maker / solver.
  • Checkpoint decisions with no consequence.
Learning Section
Flashcard Learning
Mechanical memorization of essential terms, formulas, vocabulary, definitions, symbols, anchors.
Student does
Reviews flashcards freely. No time limit. Scrolls and revisits as needed. Must view all flashcards at least once before the Memory Check.
  • Flashcards being counted as conceptual mastery.
  • Flashcards completing the learning branch without Memory Check.
Required Test
Memory Check
Quizlet-style test generated from the flashcards. Checks memorization readiness, not deep conceptual mastery.
Question types
True/False · Multiple Choice · Tile Match · Term ↔ Definition · Formula ↔ Name · Vocabulary ↔ Meaning.
Answers
Do you know the key cards? — not — Can you apply the concept deeply?
  • Memory Check replacing conceptual Practice Arc.
  • Forced order between Case Study and Flashcards. (Cross-section)
Gate
Unlock Gate
Separates learning preparation from actual homework practice. Prevents blind practice.
Unlocks when
Case-Based Preview completed with minimum acceptable result · Flashcards viewed at least once · Memory Check completed with minimum acceptable result.
UI
Chain-breaking animation. Button reads Enter Practice Arc / Start Practice Arena / Practice Arc Unlocked. Never Start Homework — student is already inside the Homework Tab.
  • Unlocking Practice Arc without both learning branches completed.
Practice Arc
Practice Arc
The actual homework body. Where games, interactivity, and simulations live. One full mission arc leading to one Boss.
Tests
Conceptual understanding · critical thinking · debugging · error detection · calculation · issue→formula conversion · proof/reasoning · simulation-based application · subject-specific applied skills.
Minimum structure
Full / hard: Practice Mission(s) → Simulation / Medium-Hard Challenge → Boss Arena.
Lighter / easy: At least one strong conceptual practice + mini-boss or final challenge.
  • Practice Arc as random disconnected games.
  • Practice Arc focusing mainly on memorization.
  • Practice tasks that do not match target skill.
Mastery Peak
Boss Arena
Mastery peak of the Practice Arc. Reasoning chain, not a quiz with HP.
Locked reasoning skeleton
Why is this formula / method needed?
How do you apply it?
What does the result mean?
Tests
Conceptual understanding · application · transfer · explanation · reasoning chain · weak points from earlier activities · integrated lesson mastery.
  • Boss as quiz + HP skin.
  • Boss without Why → How → What reasoning.
  • Boss testing only flashcard recall.
Debrief
Reflection / Marking
Short debrief plus feedback/marking input. Closure and improvement route — not a long essay or generic ritual.
Student may answer
What was hardest? · Why did you make your main decision? · What mistake would you avoid next time? · What concept finally clicked?
System outputs
Score · feedback · weak points · strong points · redo recommendation · passed / not-passed status · next-step suggestion.
Pass / retake
Use Attempt completed, homework not passed or Needs Retry. Retakes use same concepts, not exact same questions.
  • Generic Reflection disconnected from actual performance.
  • Exact-question retake farming.
  • "Not Completed" as the only label after a real attempt.

Buzan Distribution

Not a phase.
A spine.

The old standalone Consolidation phase is superseded. Buzan runs as a cross-flow foundation, with Anchor Lock behavior redistributed across Flashcards, Memory Check, Reflection, and Practice Arc when needed.

Case-Based Preview
Story association · visual hooks · simplified concept maps
Flashcard Learning
Memory anchors · key terms, formulas, vocabulary
Memory Check
Active recall · retrieval · pattern reinforcement
Practice Arc
Pattern recognition · mistake linking · applied association
Boss Arena
Why → How → What chain as the mastery reasoning pattern
Reflection
Weak-point memory recap · next anchor · learning loop closure

  • Uzbek Language Foundation. Owns language quality, simplification rules, formal register, cultural context. Generation-time contract, not cleanup.

  • Open questions remain. Scoring weights, unlock thresholds, retake scoring method, Easy vs Hard boss behavior, default Practice Arc mission count — tracked in the decision log, not here.
