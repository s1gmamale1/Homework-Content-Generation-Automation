Nine phases. One gate.
Ship canonical, export safe.

The migration playbook for turning Homework-Content-Generation-Automation into a Flow v2 generator while keeping Homeworks runtime compatible. Build full canonical Flow v2 now; export only beta-safe content until the platform can render the full flow. The chain is book → chapter/section → source map → Flow v2, and every job emits content artifacts only — five outputs, no runtime code. One platform prerequisite, eight generator PRs, fifteen stop-the-line conditions. No ghost CBP. No invalid content_json. No "v2 ready" cosplay.

03 / content artifacts & infra registry

04 / execution start

Begin two tracks in parallel.

Track A unlocks true runtime Flow v2 and removes the ghost-content risk. Track B builds the generator backbone while the platform catches up — neither needs the other finished.

TRACK A · PLATFORM PREREQUISITE
Land the runtime gate first
[Flow v2 Phase 0] Add CBP runtime and Unlock Gate
Why first: this is the lock on the castle gate. Without it the generator can create CBP, but students never see it.
TRACK B · GENERATOR FOUNDATION
Build the backbone in parallel
[Flow v2 Phase 1] Add AI Gateway and canonical schemas [Flow v2 Phase 2] Add SourceMap and prompt/standard sync
Why parallel: neither needs platform runtime finished. They lay the generation spine while Track A renders.

08 / branch / pr strategy

One PR per phase. Paper trail every contract change.

Phase completion = PR merged + tests green + result doc committed. After each phase: update the result doc, roadmap (if a contract changed), compatibility matrix, prompt/standard manifest, then re-run conformance.

MERGE ORDER + TITLE FORMAT
[Flow v2 Phase N] <deliverable name>
Platform Phase 0 ∥ Generator Phase 1–2
Phase 3+ must not claim runtime-v2 readiness until Phase 0 lands
Phase 7 must land before any default flip
REQUIRED PR CHECKLIST
  • Tests added
  • Legacy mode preserved
  • Beta export contract checked
  • Answer-leak check passed
  • Prompt/standard source recorded
  • Phase result doc committed
